Jayway JsonPath Examples

JsonPath is a query language for JSON that allows you to extract specific data from a JSON structure. Jayway JsonPath is a Java library that provides an implementation of JsonPath.

With Jayway JsonPath, you can define expressions to traverse JSON structures and retrieve data based on certain criteria. It supports a syntax similar to XPath, which is used for XML documents. JsonPath expressions can be used to filter, map, and extract data from JSON objects.

Example 1: Extracting a Single Value

        
            String json = "{ "name": "John", "age": 30, "city": "New York" }";
            Object name = JsonPath.read(json, "$.name");
            System.out.println("Name: " + name);

In this example, the JsonPath expression $.name is used to extract the value of the "name" field from the JSON document.

Example 2: Filtering with Criteria

        
            String jsonArray = "[{ "name": "Alice", "age": 25 }, { "name": "Bob", "age": 35 }]";
            List<String> names = JsonPath.read(jsonArray, "$[?(@.age > 30)].name");
            System.out.println("Names of people older than 30: " + names);

This example demonstrates filtering an array of objects based on a criteria (age > 30) and extracting the names of matching objects.
